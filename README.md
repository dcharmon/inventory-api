## Fo76 Inventory and Item Management RESTful Web Service

---

### Overview 

---

Many applications require the ability to manage collections of items such as equipment. However, inventory systems are often built in a project-specific way, with hardcoded data and limited support for categories and attributes. This makes it difficult to reuse these systems across multiple applications.

The Fo76 Inventory and Item Management API provides a RESTful service that allows developers to manage items in a structured and flexible way. The service supports item attributes and provides standardized endpoints for creating, retrieving, updating, and searching inventory data.

### This RESTful web service offers developers endpoints to integrate standardized, real-time armor statistics and manage personal stash into their build planners, trading platforms, and companion tools.

The service provides the following:

- GET a list of all armor pieces.

- GET a specific armor piece.

- GET a current inventory of a specific user.

- Search for all top-tier armor pieces.

- Add an item.

- DELETE an item.
